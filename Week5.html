Week 5: Reflectie & Beslissingen

Van Figma naar HTML & CSS.

Doelen

Het doel van deze week was om mijn ontwerp uit Figma om te zetten naar HTML en CSS. Ik wilde de basisstructuur van de website opbouwen, werken met containers en mijn gekozen kleuren en lettertypes toepassen in de code. Daarnaast wilde ik leren hoe ik mijn project online kan zetten via GitHub en Netlify.

Wat heb ik gedaan?

  • De HTML-structuur opgezet met verschillende pagina’s per week.
  • Een navigatiebalk toegevoegd zodat alle weken verbonden zijn.
  • Lettertypes uit Figma geïmporteerd via Google Fonts.
  • Een achtergrondkleur en basisopmaak toegevoegd.
  • Het project gepubliceerd via GitHub en Netlify.

Redenen voor mijn keuzes

Ik koos voor een rustig en overzichtelijk ontwerp met voldoende witruimte, in lijn met mijn Figma-ontwerp. De gele accentkleur zorgt voor consistentie tussen design en code.

Lessen & Next steps

Ik leerde hoe belangrijk een duidelijke HTML-structuur is. Volgende week wil ik experimenteren met lay-out, grid en spacing zodat de website nog professioneler oogt.
